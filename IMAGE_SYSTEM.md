# 🖼️ ระบบรูปภาพ — สรุปการทำงาน

## ลำดับความสำคัญ (Fallback Chain)

ทุกรอบโพส ระบบเลือกรูปตามลำดับนี้ — ตัวไหนได้ก่อนใช้ตัวนั้น:

```
1. รูป USER (my_images/<page>/)   ← ใช้ก่อนเสมอถ้ามี
        ↓ (ถ้าโฟลเดอร์ว่าง)
2. AI Video (Pollinations)         ← รอบ 12:00, 18:00
        ↓ (ถ้าไม่ใช่รอบ video / AI ล่ม)
3. AI Image (Pollinations)         ← AI คิด+สร้างรูปให้
        ↓ (ถ้า AI ล่ม)
4. Pexels Stock                    ← รูป stock ฟรี
        ↓ (ถ้าไม่มีรูปเลย)
5. ข้อความอย่างเดียว                ← โพสได้เสมอ
```

## แต่ละ Page ใช้อะไร

| Page | โฟลเดอร์ | ตอนนี้ใช้ |
|------|----------|-----------|
| ขายไอที (TON AI Tech) | my_images/ขายไอที/ | รูป USER (28 รูป) |
| ขนมจีนทะเลสด | my_images/ขนมจีนทะเลสด/ | AI สร้าง (ว่าง) |
| การตลาด (Poshji) | my_images/การตลาด/ | AI สร้าง (ว่าง) |
| สลัดคลีน | my_images/สลัดคลีน/ | AI สร้าง (ว่าง) |

## Auto-Resize ตาม Platform

รูปทุกแหล่ง (USER หรือ AI) ถูก resize อัตโนมัติ:

| Platform | ขนาด | อัตราส่วน |
|----------|------|-----------|
| Facebook | 1200 × 630 | แนวนอน |
| LINE | 1040 × 1040 | จตุรัส |
| TikTok | 1080 × 1920 | แนวตั้ง |

## วิธีเพิ่มรูปให้ page อื่น

แค่เอารูปใส่โฟลเดอร์ — ระบบสลับมาใช้รูป USER ทันที:
```
F:\ProjectAI\auto_poster\my_images\ขนมจีนทะเลสด\   ← ใส่รูปอาหารทะเล
F:\ProjectAI\auto_poster\my_images\การตลาด\        ← ใส่รูปการตลาด
F:\ProjectAI\auto_poster\my_images\สลัดคลีน\       ← ใส่รูปสลัด
```

ไม่ต้อง restart — รอบถัดไปใช้เลย
