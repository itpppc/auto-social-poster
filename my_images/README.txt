  วิธีใส่รูปของคุณเอง — ระบบจะดึงไปโพสอัตโนมัติ
═══════════════════════════════════════════════════════════

โครงสร้างโฟลเดอร์:

  my_images/
    ขนมจีนทะเลสด/      <- รูปสำหรับ Page ขนมจีนทะเลสด
      01.jpg
      02.jpg
      03.jpg
    การตลาด/           <- รูปสำหรับ Page Poshji Studio (การตลาด)
    ขายไอที/           <- รูปสำหรับ Page ขายไอที (TON AI Tech)
    สลัดคลีน/          <- รูปสำหรับ Page สลัด/อาหารคลีน
    _default/          <- รูปกลาง ใช้เมื่อ page อื่นไม่มีรูป

───────────────────────────────────────────────────────────

วิธีใช้:
  1. เอารูปสวยๆ ของคุณใส่ในโฟลเดอร์ตาม page
  2. ตั้งชื่อไฟล์อะไรก็ได้ (01.jpg, promo.png, ...)
  3. รองรับ .jpg .jpeg .png .webp
  4. ระบบจะสุ่มเลือกรูปที่ยังไม่ถูกใช้ → โพสอัตโนมัติ
  5. ใช้ครบทุกรูป → วนกลับมาใช้ใหม่

ลำดับความสำคัญ:
  1) รูปของคุณ (โฟลเดอร์นี้)   ← ใช้ก่อนเสมอ
  2) AI สร้างรูป (Pollinations) ← ถ้าโฟลเดอร์ว่าง
  3) Pexels stock              ← ถ้า AI ล่ม
  4) โพสข้อความอย่างเดียว       ← ถ้าไม่มีรูปเลย

───────────────────────────────────────────────────────────

ขนาดรูปแนะนำ:
  - Facebook: 1200 x 630 px (แนวนอน) หรือ 1080 x 1080 (จตุรัส)
  - รูปคมชัด สว่าง มี contrast ดี
  - ไม่ต้องใส่ข้อความบนรูปมาก (caption ระบบเขียนให้)

หลังใส่รูป:
  ไม่ต้อง restart อะไร — รอบโพสถัดไปจะดึงไปใช้เลย
